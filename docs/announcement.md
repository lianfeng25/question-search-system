## 🛠️ 使用说明

### 快速开始
1. **导入题库**：支持Excel、Feather、CSV格式
2. **导入试卷**：支持文本文件或直接粘贴
3. **执行检索**：点击按钮开始智能匹配
4. **查看结果**：在右侧查看匹配详情和答案

### 截图工具使用
1. **打开工具**：点击"智能截图"按钮
2. **选择模式**：区域截图或滚动截图
3. **选择区域**：用鼠标拖动选择截图区域
4. **保存结果**：支持保存为图片或PDF
